My Project

Project Title: AI-Based Face Recognition for Attendance

Objective: The aim of the project is to automate attendance using AI and face recognition technology, ensuring accuracy and reducing manual work.

My Role: I worked on integrating the face recognition system using Python and OpenCV, and ensured real-time detection and logging into the system.

Advantages:

  • Reduces manual attendance work
  • Prevents proxy attendance
  • Improves accuracy and saves time
